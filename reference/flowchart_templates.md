# Flow diagram templates

Registry of the bundled flow diagram templates.

## Usage

``` r
flowchart_templates
```

## Format

A data frame with columns:

- template_id:

  Template identifier.

- name:

  Display name.

- guideline_id:

  Linked guideline in
  [guidelines](https://choxos.github.io/reportilo/reference/guidelines.md).

- study_type:

  Study type the diagram applies to.

- n_count_fields:

  Number of fillable fields.
