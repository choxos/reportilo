# Risk-of-bias tool domains

The domains of each risk-of-bias tool (see
[rob_tools](https://choxos.github.io/reportilo/reference/rob_tools.md)).

## Usage

``` r
rob_domains
```

## Format

A data frame with columns:

- tool_id:

  Tool identifier.

- domain_id:

  Domain id within the tool (e.g. "D1").

- label:

  Full domain description.

- domain_order:

  Display order.

## Source

robvis and the underlying tool manuals.
