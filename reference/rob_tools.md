# Risk-of-bias assessment tools

The supported risk-of-bias tools.

## Usage

``` r
rob_tools
```

## Format

A data frame with columns:

- tool_id:

  Tool identifier (e.g. "rob2", "robins_i").

- name:

  Display name.

- study_type:

  Study type the tool applies to.

- n_domains:

  Number of domains.

- levels:

  Allowed judgment levels, "; "-separated, in order.

## Source

robvis (McGuinness & Higgins) and the underlying tool manuals.
