# Risk-of-bias judgment levels

The judgment levels used across the tools, with the color and symbol
used to render them (following robvis).

## Usage

``` r
rob_levels
```

## Format

A data frame with columns:

- level:

  Judgment level (e.g. "Low", "High", "Critical").

- color:

  Hex fill color.

- symbol:

  Symbol drawn on the traffic-light point.

- level_order:

  Severity rank for ordering summary bars.

## Source

robvis color scheme.
