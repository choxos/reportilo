# Example risk-of-bias assessments

A small example assessment for each tool, used to seed
[`reportilo_rob()`](https://choxos.github.io/reportilo/reference/reportilo_rob.md).

## Usage

``` r
rob_example
```

## Format

A data frame in long format with columns:

- tool_id:

  Tool identifier.

- study:

  Study label.

- domain_id:

  Domain id (or "Overall").

- judgment:

  The judgment level.
